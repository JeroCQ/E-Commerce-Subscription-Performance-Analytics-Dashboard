# E-Commerce-Subscription-Performance-Analytics-Dashboard

Power Query: Merging messy spreadsheets, handling null values, and unpivoting data.

Data Modeling: Creating a Star Schema, managing active/inactive relationships, and building a standardized Date Table.

DAX: CALCULATE, RELATED, Time Intelligence (SAMEPERIODLASTYEAR, YTD), and conditional metrics.

Visualization: Bookmarks, tooltips, drill-throughs, and dynamic switching between dark/light themes or currencies.
